# Sistema Bancário em Java (POO)

Projeto em **Java** desenvolvido para demonstrar conceitos fundamentais de **Programação Orientada a Objetos**, utilizando um sistema bancário simples como exemplo.

## Conceitos aplicados
- Herança (`extends`)
- Encapsulamento
- Upcasting e Downcasting
- `super` e `instanceof`

## Estrutura
- `Account` = classe base
- `BusinessAccount` = conta empresarial com empréstimo
- `SavingsAccount` = conta poupança com juros
